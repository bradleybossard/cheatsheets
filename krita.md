```
right-click / top pen button            Wheel selector

[                                       Decrease brush size
]                                       Increase brush size
Shift + drag mouse left/right           Increase/decrease brush size
= / -   (not + / -)                     Zoom in/out
left mouse + drag                       Drag canvas
x                                       Swap foreground/background color
4 / 6                                   Rotate canvas left/right
5                                       Reset rotation
```
