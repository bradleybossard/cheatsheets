```
OSX / Win or Linux

cmd+o / ctrl+n                 # Open class
cmd+shift+o / ctrl+shift+n     # Open file
cmd+alt+o   / alt+shift+n      # Open symbol
# after opening a file, append :<line-number> to jump to specific line

cmd+e / ctrl+e                 # Recently opened files
cmd+shift+e / ctrl+shift+e     # Recently edited files
cmd+alt+←/→ / ctrl+alt+←/→
cmd+shift+backspace  / ctrl+shift+backspace   # Jump to last edit location

alt+f7 / alt+f7                # Show usage of symbol
cmd+alt+f7 / ctrl+alt+f7       # Show usage of symbol in place

cmd+b / ctrl+b                 # Goto declaration
cmd+click / ctrl+click         # Goto declaration
cmd+alt+b / ctrl+alt+b         # Goto implementation

cmd+u / ctrl+u                 # Goto superclass
```
